## Project Overview

This project is a modern, elegant front-end landing page for a fictional premium sweater e-commerce brand called WOOL HAVEN, meticulously built using HTML5 and CSS3. The primary goal was to create a sophisticated fashion retail interface with strong typography hierarchy, minimalist design aesthetics, and seamless responsiveness across various devices. This project served as an excellent opportunity to demonstrate modern web design principles and advanced CSS layout techniques.

## Features

* **Sophisticated Typography Hierarchy:** Expertly paired Playfair Display, Montserrat, and Poppins fonts to create visual contrast and establish brand identity.
* **Responsive Design:** Implemented using modern CSS techniques (Flexbox and CSS Grid) to ensure seamless adaptability and optimal viewing on desktop, tablet, and mobile screens.
* **Semantic HTML5:** Structured content using appropriate HTML5 elements for better accessibility and SEO.
* **Product Showcase Grid:** Clean layout displaying curated sweater collections with professional product photography.
* **Minimalist Aesthetic:** Black, white, and beige color palette conveying premium brand perception.

## Technologies Used

* **HTML5:** For semantic markup and content structure.
* **CSS3:** For styling, layout Flexbox, typography, and responsive design.
* **Google Fonts:** Montserrat, Playfair Display, and Poppins for professional typography.
